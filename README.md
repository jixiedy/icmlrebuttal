# ICML Rebuttal

This repository contains the rebuttal materials for our ICML submission.

## Structure

- `rebuttal.md` — Point-by-point responses to all reviewer comments

## How to Use

The rebuttal addresses each reviewer's concerns in order. Each response is prefixed with the reviewer number and concern ID (e.g., **R1Q1** for Reviewer 1, Question/Concern 1).
